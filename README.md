# UTS PAM
Di dalam repository ini terdapat sebuah aplikasi sederhana berupa kalkulator dengan nama Calc.</br></br>
Pada aplikasi ini terdapat :</br>
- 10 bilangan {1,2,3,4,5,6,7,8,9,0}</br>
- 1 koma('.')</br></br>
Serta beberapa fungsi :</br>
- Penjumlahan('+')</br>
- Pengurangan('-')</br>
- Perkalian('*')</br>
- Pembagian('/')</br>
- *Delete*("DEL")</br>
- *Clear*('C')</br>
- Samadengan('=')</br></br>
